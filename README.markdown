Random Haskell scripts too tiny to deserve their own repo.

- `bfi.hs`: `mtl`
- `factorize.hs`: `arithmoi`
- `mycal.hs`: `split`, `time`
- `numeral.hs`: `numerals`
- `sortchar.hs`: `split`
- `transpose.hs`: `cmdargs`
- `unamp.hs`: `attoparsec` (and therefore `bytestring`)
